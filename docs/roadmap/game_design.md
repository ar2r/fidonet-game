# Game Design Concept: FidoNet 1995 Simulator

## 1. Концепция (Concept)
**Жанр**: Text-based Puzzle / Simulator / RPG
**Сеттинг**: 1995 год, постсоветское пространство, ночь, гудит модем.
**Цель**: Пройти путь от "поинта" (Point) до Сисопа (SysOp) и, возможно, Координатора (NC/RC).
**Визуальный стиль**: Строгий TUI (Text User Interface). Цветовая гамма: Синий фон/Белый текст (Norton Commander, GoldED), Черный терминал/Зеленый текст (Modem terminal). ASCII-арт.
**Язык**: Русский.

## 2. Сценарий (Scenario)

### Акт 1: Коннект (The Connect)
Игрок начинает перед "черным экраном" терминала.
- **Задача**: Настроить модем и дозвониться до первой BBS, чтобы скачать софт для Фидо.
- **Геймплей**: Ввод AT-команд (ATZ, ATDP...), подбор строки инициализации, слушание звуков модема (текстовое описание: *BZZZZT... KRRR...*).
- **Успех**: Получение файлов `T-Mail` и `GoldED`.

### Акт 2: Настройка станции (The Setup)
Интерфейс меняется на "синий" (стиль Norton Commander / редакторов конфигов).
- **Задача**: Настроить `t-mail.ctl` и `golded.cfg`.
- **Геймплей**: Пазл с подбором правильных флагов, путей и адресов. Ошибки приводят к "Unrecognized lookup" или зависанию "тоссера".
- **Событие**: Получение первого поинт-адреса (например, `2:5020/123.45`).

### Акт 3: Первая почта (The Mail)
Основной геймплей в интерфейсе GoldED.
- **Задача**: Прочитать правила эхи `SU.FLAME`, не нарушить их, и правильно ответить.
- **Механики**:
    - Навигация по письмам (стрелочки, плюсик для раскрытия тредов).
    - "Квотинг" (цитирование): Игрок должен правильно обрезать цитаты (оверквотинг карается).
    - Выбор правильных ответов в диалогах (RPG-элемент).
- **Конфликт**: Вас троллят деды фидошники. Нужно не сорваться (или сорваться красиво).

### Акт 4: Апгрейд (The Upgrade)
- **Задача**: Стать нодой.
- **Геймплей**: Решение задач на социальную инженерию (уговорить босса выделить телефонную линию в офисе под ночную станцию).

## 3. Технические Требования (Technical Requirements)

### Frontend Engine
- **Стек**: React или Vue (для компонентного подхода к UI окнам).
- **Стилизация**:
    - Моноширинные шрифты (Terminus, DOS VGA).
    - CSS Scanlines / CRT effect (опционально, отключаемо).
    - Строгое ограничение сетки (80x25 символов).

### State Management (The "Link Save")
Весь прогресс должен сериализоваться в URL.
- **Данные**:
    - Текущий уровень / Акт.
    - Инвентарь (какой софт скачан).
    - Статус (карма, адрес).
    - Лог последних действий.
- **Метод**: `JSON` -> `MessagePack` (или `LZString`) -> `Base64` -> `Hash Fragment`.
- **Пример**: `fidonet.game/#/save/N4IgdghgtgpiBcIDiAaeA...`

## 4. UI Референсы (UI References)
1.  **Terminal**: Черный фон, мигающий курсор `_`.
2.  **GoldED**: Верхняя полоска меню, нижняя полоска статуса клавиш (F1 Help, F2 Save...), список писем, окно просмотра.
3.  **Setup/Config**: Текстовые поля ввода, чекбоксы `[x]`.

## 5. Идеи для контента и "фишек" (Ideas)
- **Звук**: Эмуляция звука модема при "дозвоне" (Web Audio API).
- **Пасхалки**: Известные фидошные имена, локальные мемы (кащенизм, плюсомет).
- **Тайминг**: Реальное ожидание "ZMH" (Zone Mail Hour). Игрок не может отправить почту, пока не наступит "ночь" (или можно пропустить время).

## 6. Роли и Развилки (Roles & Paths)
Игра предлагает выбор пути в начале или в процессе игры:

### Путь Поинта (The Point Path)
*Слоган: "Read Only? No way."*
- **Цель**: Получить поинт, писать в эхи, качать файлы, не быть отключенным.
- **Геймплей**: Больше про контент, общение, поиск файлов.
- **Квесты**:
    1.  **"Policy Quiz"**: Пройти тест на знание Политики у строгого Сисопа, чтобы получить адрес.
    2.  **"ZModem Resume"**: Скачать 2MB файл (Doom 2 wad) на линии, которая рвется каждые 5 минут. Нужно вовремя жать "Resume".
    3.  **"The Flamer"**: Выиграть спор в `SU.POL` и не получить `[!]` от модератора.

### Путь Сисопа (The SysOp Path)
*Слоган: "Exonia is my home."*
- **Цель**: Поддерживать аптайм, растить нодлист, бороться с ламерами.
- **Геймплей**: Больше про конфиги, железо, администрирование.
- **Квесты**:
    1.  **"The Night Call"**: 3 часа ночи. Звонок голосом на модемную линию. Нужно быстро снять трубку и гаркнуть "DATA!!!", чтобы факс на той стороне не отвалился, или поговорить с пьяным юзером.
    2.  **"Boss Fight (Literal)"**: Шеф увидел счет за межгород. Нужно придумать оправдание (это все вирусы / это нужно для бизнеса).
    3.  **"Tosser Tuning"**: Настроить тоссер так, чтобы почта улетала за 1 минуту льготного времени.
    4.  **"Leecher Ban"**: Вычислить "пиявку", который только качает файлы и ничего не пишет, и красиво его "отлучить".

## 7. Вопросы к пользователю (Questions)
1.  **Глубина симуляции**: Насколько хардкорным должен быть ввод команд?
    - *Вариант А*: Полностью текстовый ввод (как в Linux консоли).
    - *Вариант Б*: Выбор из вариантов ответов (квестовый стиль), но выглядит как консоль.
    - *Рекомендация*: Вариант Б для играбельности, но с возможностью вводить "секретные" чит-коды руками.

2.  **Сюжетный тон**:
    - *Серьезный*: Историческая реконструкция.
    - *Юмористический*: Гипертрофированные образы сисопов, мемы.

3.  **Мобильная версия**:
    - Нужна ли адаптация под телефон? (Интерфейс 80x25 очень мелок на телефоне, придется делать скролл или зум).
