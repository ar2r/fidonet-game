# Оценка предложений улучшения геймплея

**Дата:** 2026-02-09

## Текущее состояние игры

- **Реализовано**: Акты 1-2 (полный геймплей: ATZ → DIAL → Download → T-Mail config → GoldED config → Mail tossing)
- **Определены, но НЕ реализованы**: Акты 3-6 (квестовые дефиниции есть, UI/геймплей — нет)
- **Ключевой блокер**: нет GoldED-читалки (чтение/написание писем) — без неё невозможен Акт 3+

## Рекомендуемый план внедрения

### Волна 1: Quick Wins + критический путь к Акту 3

**Цель**: сделать игру играбельной до Акта 3 включительно

1. **История команд (Up/Down)** — массовое улучшение UX
2. **Tab-автодополнение** — список известных команд
3. **GoldED Reader** — основная задача. TUI для чтения/навигации по эхоконференциям. Без этого Акт 3 невозможен
4. **Система писем/сообщений** — генерация контента для SU.FLAME (правила, тролль, NPC-посты)
5. **Написание ответа в эху** — форма compose/reply для квестов `read_rules` и `reply_welcome`/`trace_troll`

### Волна 2: Экономика + Акт 4

**Цель**: работающая экономика и переход в ноду

6. **Radio Market (магазин)** — TUI с покупкой железа. Нужен для квеста `hardware_upgrade`
7. **Скиллы с эффектами** — typing влияет на опечатки, hardware открывает покупки, eloquence влияет на диалоги
8. **Телефонные счета** — ежемесячный платёж, зависящий от онлайн-времени
9. **Диалоговая система (NPC чат)** — для квеста `request_node` (разговор с сисопом)
10. **BinkleyTerm Config TUI** — по аналогии с T-Mail/GoldED конфигураторами
11. **ZMH Uptime механика** — квест `nightly_uptime`

### Волна 3: Конфликт и кульминация (Акт 5)

**Цель**: развилки и финал

12. **Флейм-войны** (упрощённая версия) — выбор аргументов из списка, а не полная карточная система
13. **Случайные события** — мама подняла трубку, звонок друга, обрыв связи
14. **Акт 5: Кризис** — техническая или дипломатическая ветка

### Волна 4: Polish и retention

**Цель**: глубина и реиграбельность

15. **Аудио** (звук модема, HDD) — Web Audio API
16. **Коллекционирование** (вирусная лаборатория)
17. **Доска объявлений (фриланс)**
18. **X-Files пасхалки** (Призрачная BBS)
19. **SysOp Tycoon** (эндгейм)

## Резюме

**Главный приоритет** — GoldED Reader и система сообщений. Это единственный блокер, не дающий игре перейти к Акту 3. Всё остальное выстраивается вокруг этого.

Quick wins (история команд, автодополнение) можно делать параллельно — они улучшают текущий опыт.

Экономику и скиллы лучше внедрять вместе с Актом 4, где они становятся значимы (магазин, покупка модема).

Флейм-войны и SysOp Tycoon — самые амбициозные фичи. Их стоит упрощать: карточные битвы → выбор из 3-4 аргументов; управление BBS → простой менеджер с текстовыми событиями.
