# Вступление: Первое Подключение (The First Connection)

## Концепция
Игрок начинает игру в роли новичка (User), который только что собрал свой первый компьютер или купил модем. Действие происходит в середине 90-х (1995-1996 гг.).

## Сюжетная завязка (The Hook)
На рабочем столе компьютера игрока, помимо системных иконок, лежит текстовый файл `README.TXT` (или `NOTE.TXT`).
Это записка от друга (Алекс), который и "подсадил" героя на тему сетей.

**Содежимое NOTE.TXT:**
```text
Привет! Поздравляю с покупкой модема!
Как и обещал, вот номер локальной BBS, где сидит нормальный Сисоп:
555-3389 (The Nexus BBS)
Скажи ему, что ты от меня, он даст тебе поинта.
Вход через терминал. Не забудь команды модема!
Удачи в Сети!

-- Алекс
```

## Механика Первого Взаимодействия

1.  **Исследование Рабочего Стола**:
    *   Игрок видит рабочий стол Windows 95 / ReactOS.
    *   Видит файл `NOTE.TXT`. Открывает его (двойной клик), читает записку, узнает номер телефона.
    *   Видит ярлык `Fido.bat` (или `Terminal`).

2.  **Запуск Терминала**:
    *   Игрок запускает Терминал.
    *   Видит приглашение командной строки `MODEM READY`.

3.  **Набор Номера**:
    *   Игрок должен ввести команду для набора номера.
    *   Реалистичный вариант: `ATDT 555-3389`
    *   Упрощенный вариант (если игрок тупит): `DIAL 555-3389` или `HELP`.
    *   Графическая подсказка: Стикер на мониторе с основными командами (`ATZ` - сброс, `ATDT` - набрать...).

4.  **Процесс Соединения**:
    *   Звук набора номера (писк тонального набора).
    *   Звук "хендшейка" (шипение модема).
    *   Текст в терминале:
        ```
        DIALING 555-3389...
        CONNECT 14400
        ...
        Welcome to THE NEXUS BBS
        Verifying Connection...
        Login: _
        ```

5.  **Регистрация**:
    *   Игрок вводит имя (никнейм).
    *   Система спрашивает: "New User? (Y/N)".
    *   Игрок отвечает `Y`.
    *   Заполняет короткую анкету (Ник, Реальное имя, Откуда узнал - тут можно выбрать "От Алекса").

6.  **Первый Квест**:
    *   Сисоп (NPC) приветствует в чате.
    *   "А, от Алекса? Ну проходи. Правила читал? (Y/N)".
    *   Выдает первый доступ (уровень Guest) и задание: "Прочитай Rules.txt в файловой области и скачай эхопроцессор".

## Техническая Реализация (MVP)

1.  **Компонент `TextFileViewer`**: Для отображения `NOTE.TXT` в стиле Notepad.
2.  **Обновление `TerminalWindow`**:
    *   Поддержка парсинга команд `ATDT`, `DIAL`.
    *   Симуляция задержки соединения и вывода текста построчно.
    *   Стейт-машина внутри терминала: `IDLE` -> `DIALING` -> `CONNECTING` -> `CONNECTED_LOGIN`.
3.  **Обновление `App.jsx`**:
    *   Добавить иконку `NOTE.TXT`.
    *   Добавить "Стикер" на рамку монитора (опционально, CSS).

## Атмосфера
*   Звуки модема – критически важны для погружения.
*   Мерцающий курсор.
*   Медленная отрисовка текста (эмуляция скорости 14400 бод).
