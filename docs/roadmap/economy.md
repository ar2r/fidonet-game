# Экономика и Инвентарь (Economy & Inventory)

Этот документ описывает экономическую модель игры, ресурсы и систему инвентаря.

## 1. Экономика (The Economy)

Экономика в игре построена на дефиците ресурсов и необходимости поддерживать баланс между "реальными" деньгами и "виртуальным" статусом.

### 1.1. Валюты

1.  **Рубли (RUR)**
    *   *Назначение*: Оплата телефона (критично!), покупка железа, еды, такси до радиорынка.
    *   *Источник*: Карманные деньги, подработки, перепродажа.
2.  **Доллары ($)**
    *   *Назначение*: Покупка элитного импортного железа (US Robotics Courier) и лицензионного софта (редкость).
    *   *Источник*: Обменники, продажа редких запчастей "новым русским".
3.  **Пиво / Сок / Шоколад**
    *   *Назначение*: "Жидкая валюта". Взятки сисопам, подарок друзьям, восстановление *Sanity*.
    *   *Источник*: Магазин, холодильник.
4.  **Credits (BBS Ratio)**
    *   *Назначение*: Виртуальная валюта на BBS. Тратится на скачивание (Download).
    *   *Источник*: Загрузка файлов (Upload), бонус за чат.

### 1.2. Доходы и Расходы

| Источник Дохода | Риск / Затраты | Пример |
| :--- | :--- | :--- |
| **Pocket Money** | Нет | Родители дают 100р в неделю. Если оценки в школе плохие — 0р. |
| **PC Repair** | Время, *Hardware* Skill | Сосед попросил переустановить "Винду". Награда: 50р. |
| **Trading** | Деньги | Купить коробку дискет оптом, продать в розницу в школе. |
| **Hacking** | *Software*, Карма | Взлом аккаунтов на платных BBS (опасно!). |

| Статья Расходов | Описание | Последствия неуплаты |
| :--- | :--- | :--- |
| **Телефонный счет** | Ежемесячный платеж. Зависит от времени в онлайне. | Отключение линии -> Game Over (или квест "Помириться с родителями"). |
| **Железо** | Апгрейды ПК. | Нельзя запустить новый софт или игру. |
| **Расходники** | Дискеты, бумага для принтера. | Некуда записать скачанный варез. |
| **Еда** | Пицца, сники. | Падение энергии, дрожание рук (Typing debuff). |

## 2. Инвентарь (Inventory)

Инвентарь разделен на **Физический (Рюкзак/Комната)** и **Виртуальный (HDD)**.

### 2.1. Физический Инвентарь (Physical Slots)

Игрок экипирует свой ПК.

*   **CPU**: 286 -> 386 -> 486 -> Pentium. (Влияет на скорость работы софта).
*   **RAM**: 1MB -> 4MB -> 16MB. (Нужно для запуска Windows и Doom).
*   **HDD**: 40MB -> 200MB -> 1GB. (Лимит на хранение почты и файлов).
*   **Modem**:
    *   *No-name 2400*: Глючит, рвет связь.
    *   *ZyXEL U-1496*: Хорошо держит линию, есть факс.
    *   *US Robotics Courier*: Мечта фидошника. Максимальная скорость.
*   **Keyboard**:
    *   *Мембранка*: Обычная.
    *   *Механика (IBM Model M)*: Бонус к *Typing* +10%. Громкий звук (Aggro родителей +20%).

### 2.2. Расходники (Consumables)
*   **Дискета 3.5"**: Используется для переноса файлов к друзьям (квесты) и бекапов. Имеет "здоровье" (Bad blocks).
*   **Книга "Фидонет для чайников"**: Одноразовое прочтение дает XP к навыку *Software*.
*   **Кофе**: +Energy, -Health.

### 2.3. Виртуальный Инвентарь (Software)
Хранится на HDD. Занимает место.

*   **Mailer**: BinkleyTerm, T-Mail.
*   **Editor**: GoldEd, Msged.
*   **Games**: Doom 2, Prince of Persia (для восстановления *Sanity*).
*   **Viruses**: `OneHalf.3544` (можно случайно подцепить, нужно лечить антивирусом).
*   **Warez**: Скачанные архивы. Можно распаковать или записать на дискету.
