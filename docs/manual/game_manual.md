# FidoNet Simulator Manual

> **Welcome to the year 1995.**
> You are a SysOp (System Operator) running your own BBS (Bulletin Board System) from your bedroom.
> The goal is simple: Build the best node in the net, upgrade your hardware, and become a Point or even a Coordinator.

## Vital Statistics

### 1. Sanity (Mental)
Running a BBS is stressful. Trolls, glitches, and bad connects drain your sanity.

### 2. Mom's Patience
You live with your parents. You use the family phone line.
*   **Game Over**: If patience hits 0, she pulls the plug.

## Basic Commands

| Command | Description |
| :--- | :--- |
| `HELP` / `MANUAL` | Show this help file. |
| `AT...` | Modem commands. |
| `EXIT` | Close connection. |
