# Прохождение: Симулятор FidoNet 1995

Подробное пошаговое руководство по прохождению всех этапов игры.

---

## Общая информация

Игра разделена на **3 акта** (реализованных), каждый из которых содержит несколько квестов. Прогресс линейный, за исключением Акта 3, где игрок выбирает один из двух путей.

**Полезные команды на любом этапе:**
| Команда | Действие |
|---------|----------|
| `HELP` / `MANUAL` | Показать справку (разная в DOS и в терминале) |
| `HINT` / `QUEST` | Подсказка по текущему квесту |
| `DIR` / `LS` | Список файлов в каталоге |
| `CD <путь>` | Сменить каталог |
| `TYPE <файл>` | Прочитать файл |
| `CLS` / `CLEAR` | Очистить экран |
| `WORK` | Подработка (300-600 руб., -10% рассудка) |
| `ALLOWANCE` | Попросить карманные деньги (по понедельникам) |
| `PAY` | Оплатить счёт за телефон |

## Акт 1: Коннект

**Цель:** Выйти в сеть, подключиться к BBS, скачать необходимый софт.

### Квест 1.1: Строка инициализации

> *Для выхода в сеть необходимо подготовить оборудование. Запустите терминал и инициализируйте модем.*

**Награда:** Навык «Набор текста» +1

**Пошаговое прохождение:**

1. В командной строке DOS введите:
   ```
   C:\> TERMINAL
   ```
2. Запустится терминальная программа TERMINAL v3.14. Вы увидите приветствие и приглашение `TERMINAL>`.
3. Введите команду инициализации модема:
   ```
   TERMINAL> ATZ
   ```
4. Модем ответит `OK` — квест завершён.

**Подсказки из игры:**
- Модемы управляются AT-командами
- ATZ — команда сброса/инициализации
- Альтернативная команда: `AT&F` (factory reset)

---

### Квест 1.2: Первый Контакт

> *Найден номер местной BBS "The Nexus" (555-3389). Попробуйте установить соединение.*

**Награда:** Навык «Набор текста» +1

**Пошаговое прохождение:**

1. Убедитесь, что модем инициализирован (квест 1.1 выполнен).
2. Находясь в терминале, наберите номер BBS:
   ```
   TERMINAL> DIAL 555-3389
   ```
3. Начнётся анимация соединения:
   ```
   НАБОР НОМЕРА 5553389...
   СОЕДИНЕНИЕ 14400
   ```
4. Появится баннер BBS «The Nexus» и главное меню.

**Альтернативные команды набора:**
- `ATDT 555-3389` (тоновый набор)
- `ATDP 555-3389` (импульсный набор)

**Главное меню BBS:**
| Клавиша | Раздел |
|---------|--------|
| `F` | Файловая область |
| `M` | Почтовая область (недоступна до настройки GoldED) |
| `C` | Чат с Сисопом (Архитектор) |
| `G` | Выход (Goodbye) |

---

### Квест 1.3: Необходимый софт

> *Для работы в сети FidoNet вам понадобятся почтовый мейлер и редактор сообщений. Найдите их на BBS.*

**Награда:** Навык «Набор текста» +1, Навык «Софт» +1

**Пошаговое прохождение:**

1. Из главного меню BBS нажмите `F` — войдёте в файловую область.
2. Появится список файлов:
   ```
   1. T-Mail v2605       [FIDO]   45KB
   2. GoldED 2.50+       [FIDO]   78KB
   3. DOOM2.WAD          [GAME]  4.2MB
   4. AIDSTEST v1.03     [UTIL]   42KB
   5. BinkleyTerm 2.60   [NODE]  200KB
   Q. Назад в меню
   ```
3. Введите `1` — начнётся загрузка T-Mail (анимация прогресс-бара).
4. После завершения введите `2` — загрузится GoldED.
5. Квест завершается автоматически, когда оба файла скачаны.

**Что устанавливается:**
- `C:\FIDO\T-MAIL.EXE` — почтовый мейлер
- `C:\FIDO\T-MAIL.CTL` — конфиг T-Mail (пустой шаблон)
- `C:\FIDO\GOLDED.EXE` — редактор сообщений
- `C:\FIDO\GOLDED.CFG` — конфиг GoldED (пустой шаблон)

**Опциональные загрузки:**
- `3` — DOOM2.WAD (содержит вирус «Курочка Ряба»!)
- `4` — AIDSTEST (антивирус, лечит вирус из DOOM)
- `5` — BinkleyTerm (понадобится в будущем)

**Завершает Акт 1.**

## Акт 2: Настройка

**Цель:** Настроить T-Mail и GoldED с корректными сетевыми параметрами.

### Квест 2.1: Настройка почты (T-Mail)

> *Почтовый мейлер требует настройки перед использованием. Введите сетевые параметры вашей станции и босса.*

**Награда:** Навык «Софт» +2, Навык «Набор текста» +1

**Пошаговое прохождение:**

1. Откройте настройку T-Mail одним из способов:
   - Дважды щёлкните иконку **T-Mail** на рабочем столе
   - Или в терминале: `T-MAIL SETUP`
2. Откроется синий TUI-редактор конфигурации (стиль DOS).
3. Заполните поля **точными значениями:**

| Поле | Значение | Пояснение |
|------|----------|-----------|
| FidoNet Address | `2:5020/123.45` | Ваш адрес пойнта (с точкой!) |
| Session Password | `NEXUS95` | Пароль на сессию |
| Boss Node Address | `2:5020/123` | Адрес босс-ноды (без точки!) |
| Boss Node Phone | `5553389` | Только цифры, без дефисов |
| Inbound Directory | `C:\FIDO\INBOUND` | Каталог входящей почты |
| Outbound Directory | `C:\FIDO\OUTBOUND` | Каталог исходящей почты |

4. Навигация: **Tab** или **Enter** — следующее поле.
5. Нажмите **F2** для сохранения.

**Важно — правила валидации:**
- Адрес пойнта **обязательно с точкой** (формат `Z:NNNN/NNN.PP`)
- Адрес босс-ноды **без точки** (формат `Z:NNNN/NNN`). Если указать поинтовый адрес — будет ошибка «Босс-нода не может быть поинтом»
- Телефон — **только цифры**, минимум 7 символов
- Пути — формат DOS (`C:\...`)
- Все значения должны **точно совпадать** с ответами — иначе квест не засчитается

### Квест 2.2: Личность в сети (GoldED)

> *Настройте редактор сообщений. Сеть должна знать ваше имя и адрес для правильной подписи писем.*

**Награда:** Навык «Софт» +2, Навык «Набор текста» +1

**Пошаговое прохождение:**

1. Откройте настройку GoldED одним из способов:
   - Дважды щёлкните иконку **GoldED** на рабочем столе
   - Или в терминале: `GOLDED SETUP`
2. Откроется синий TUI-редактор.
3. Заполните поля:

| Поле | Значение | Пояснение |
|------|----------|-----------|
| Username | Любое имя | Ваш псевдоним в сети |
| Address | `2:5020/123.45` | Тот же адрес, что в T-Mail |
| Origin | Любой текст | Ваша «визитная карточка» в конце писем |

4. Нажмите **F2** для сохранения.
5. После сохранения автоматически запустится **анимация обмена почтой** (Mail Tossing, 8-10 секунд).

**Минимальные требования для прохождения:**
- Заполнены поля Username и Address (оба обязательны)
- Address — корректный FidoNet-адрес

**Завершает Акт 2.** После анимации обмена почтой начинается Акт 3.

## Акт 3: Почта и Флейм

**Цель:** Обменяться почтой, прочитать правила эхи, выбрать стратегию в конфликте.

### Квест 3.1: Первая Прозвонка

> *Все системы настроены. Необходимо инициировать сеанс связи с боссом для получения свежей почты.*

**Награда:** Навык «Софт» +1, Предмет «Почтовый пакет»

**Пошаговое прохождение:**

1. В командной строке DOS введите:
   ```
   C:\> POLL
   ```
   или
   ```
   C:\> T-MAIL POLL
   ```
2. Запустится анимация обмена почтой (Mail Tossing, 8-10 секунд).
3. После завершения анимации квест завершён автоматически.

---

### Квест 3.2: Устав монастыря

> *Прежде чем писать, изучите правила эхоконференции SU.FLAME.*

**Награда:** Навык «Чтение» +1, Рассудок +5

**Пошаговое прохождение:**

1. Запустите GoldED:
   - Дважды щёлкните иконку **GoldED** на рабочем столе
   - Или в терминале: `GOLDED`
2. Откроется GoldED с чёрным фоном (стиль Norton Commander).
3. В списке эхоконференций вы увидите:
   ```
    # Описание                             Сообщ  Новых   Эха
    1 Личная почта                             0      0   NETMAIL
    2 Общее общение                            7      7   SU.GENERAL
    3 Споры и флейм                            7      7   SU.FLAME
   ```
4. Стрелками ↑↓ выберите **SU.FLAME** и нажмите **Enter**.
5. Откроется список сообщений. Первое сообщение — **«Rules v1.0 (ЧИТАТЬ ВСЕМ!)»** от Moderator.
6. Выберите его и нажмите **Enter** для чтения.
7. Квест завершается автоматически при чтении сообщения с правилами.

**Управление GoldED:**
| Клавиша | Действие |
|---------|----------|
| ↑ / ↓ | Навигация по списку |
| Enter | Выбрать область / Прочитать сообщение |
| `n` или Insert | Создать новое сообщение |
| `r` или Insert (в просмотре) | Ответить на сообщение |
| Escape | Назад |
| Ctrl+Enter | Отправить сообщение (в редакторе) |

**Содержание правил SU.FLAME:**
1. Запрещён мат
2. Оверквотинг карается
3. Никакой политики
4. Запрещено обсуждение модератора

---

### Квест 3.3: Выбор Стратегии

> *В эхе SU.FLAME разгорается конфликт. Вам предстоит решить, какую сторону занять: дипломатии или силы.*

**Награда:** Нет (переходный квест)

**Пошаговое прохождение:**

1. Подключитесь к BBS:
   ```
   TERMINAL> ATZ
   TERMINAL> DIAL 555-3389
   ```
2. Из главного меню нажмите `C` — Чат с Сисопом.
3. Нажмите Enter, чтобы начать разговор.
4. Архитектор спросит:
   ```
   Архитектор: Видел, что творится в SU.FLAME?
   Troll.Master совсем озверел. Что думаешь делать?

     1. Попробую с ним поговорить. (Путь Дипломата)
     2. Вычислю его адрес и сдам. (Путь Технаря)
   ```
5. Выберите **1** или **2** — это определит следующий квест.

---

### Квест 3.4a: Глас Разума (Путь Дипломата)

> *Попробуйте успокоить участников конференции вежливым приветствием.*

**Доступен при выборе варианта 1 в квесте 3.3.**

**Награда:** Навык «Красноречие» +3, Fido-слава +2, Карма +5

**Пошаговое прохождение:**

1. Запустите GoldED (иконка на рабочем столе или команда `GOLDED`).
2. Выберите эхоконференцию **SU.FLAME**.
3. Нажмите **`n`** (или Insert) для создания нового сообщения.
4. Заполните поля:
   - **To:** `All` (или конкретный адресат)
   - **Subj:** Любая тема
   - **Body:** Напишите любое сообщение
5. Нажмите **Ctrl+Enter** для отправки.
6. Квест завершён — сообщение опубликовано в SU.FLAME.

**Альтернатива:** Вместо нового сообщения можно **ответить** на существующее — откройте любое сообщение и нажмите `r`.

**Завершает Акт 3** (путь Дипломата).

---

### Квест 3.4b: Охота на Тролля (Путь Технаря)

> *Тролль скрывается за поддельными адресами. Вычислите его настоящий узел.*

**Доступен при выборе варианта 2 в квесте 3.3.**

**Награда:** Навык «Софт» +3, Fido-слава +1

**Пошаговое прохождение:**

1. Выйдите из BBS (нажмите `G`), если вы ещё подключены.
2. В командной строке DOS введите:
   ```
   C:\> TRACE TROLL.MASTER
   ```
3. Запустится трассировка маршрута (анимация ~2 секунды):
   ```
   Tracing route to TROLL.MASTER...
    1  2:5020/730 (MyNode)  0ms
    2  2:5020/123 (The Nexus)  20ms
    3  2:5020/666 (DarkSide)  45ms
   Target found! Address: 2:5020/666
   Log saved to disk.
   ```
4. Квест завершён автоматически при успешной трассировке.

**Важно:** Команда `TRACE` работает только из DOS (режим IDLE), а не из терминала или BBS.

**Завершает Акт 3** (путь Технаря).

## Опциональные механики

### Вирус «Курочка Ряба»

Если скачать и запустить DOOM2 (`DOOM2` в командной строке), активируется вирус:
- Экран заполняется каскадными символами (эффект «Cascade»)
- Рассудок -20, Атмосфера -10
- **Лечение:** запустите `AIDSTEST` (антивирус должен быть скачан с BBS)

### Экономика

- **WORK** — подработка у соседа (переустановка Windows). Заработок 300-600 руб., но -10% рассудка.
- **ALLOWANCE** — карманные деньги от родителей. Работает только по понедельникам и только если атмосфера > 20.
- **PAY** — оплата счёта за телефон (если есть задолженность).

### Файловая система

В игре эмулируется DOS-файловая система. Полезные каталоги:
```
C:\
├── FIDO\           — FidoNet-софт и конфиги
│   ├── T-MAIL.EXE
│   ├── T-MAIL.CTL  — конфиг (можно прочитать через TYPE)
│   ├── GOLDED.EXE
│   ├── GOLDED.CFG
│   ├── INBOUND\    — входящая почта
│   └── OUTBOUND\   — исходящая почта
├── GAMES\          — игры
│   └── DOOM2.WAD   (если скачан)
└── UTILS\          — утилиты
    └── AIDSTEST.EXE (если скачан)
```

## Дерево прохождения

```
Акт 1: Коннект
  └── 1.1 Строка инициализации (ATZ)
       └── 1.2 Первый Контакт (DIAL 555-3389)
            └── 1.3 Необходимый софт (скачать T-Mail + GoldED)

Акт 2: Настройка
  └── 2.1 Настройка почты (T-Mail конфиг → F2)
       └── 2.2 Личность в сети (GoldED конфиг → F2)

Акт 3: Почта и Флейм
  └── 3.1 Первая Прозвонка (POLL)
       └── 3.2 Устав монастыря (прочитать Rules в SU.FLAME)
            └── 3.3 Выбор Стратегии (диалог с Архитектором)
                 ├── 3.4a Глас Разума (написать в SU.FLAME)     ← Путь Дипломата
                 └── 3.4b Охота на Тролля (TRACE TROLL.MASTER)  ← Путь Технаря
```

## Отладка / Debug

В меню «Пуск» есть красная кнопка **«[DEBUG] Пропустить квест»** — она позволяет пропустить текущий квест и перейти к следующему. Полезно для тестирования поздних этапов игры без полного прохождения.
